\subsection{Detailed Description}
A class to handle the Open\-G\-L calls for an image processing application. 

A single texture is used and it is operating on using G\-L\-S\-L (shaders). The class is designed to process a U\-I\-Image of a byte array image on the graphics chip easily. \begin{DoxyAuthor}{Author}
Stuart Lynch \href{mailto:stuart.lynch@uea.ac.uk}{\tt stuart.\-lynch@uea.\-ac.\-uk} 
\end{DoxyAuthor}
