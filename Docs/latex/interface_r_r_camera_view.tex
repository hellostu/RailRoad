\subsection{Detailed Description}
A class to handle the displaying of the processed camera frames. 

This class extends the U\-I\-View class (from the U\-I\-Kit). This view has an Open\-G\-L layer, so displays whatever is drawn by the active Open\-G\-L session. The class also handles an \hyperlink{interface_r_r_camera_capture_session}{R\-R\-Camera\-Capture\-Session}, which deals with pulling the image frames from the camera. \begin{DoxyAuthor}{Author}

\end{DoxyAuthor}
