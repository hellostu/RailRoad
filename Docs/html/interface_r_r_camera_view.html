Detailed Description

A class to handle the displaying of the processed camera frames.

This class extends the UIView class (from the UIKit). This view has an OpenGL layer, so displays whatever is drawn by the active OpenGL session. The class also handles an RRCameraCaptureSession, which deals with pulling the image frames from the camera.
