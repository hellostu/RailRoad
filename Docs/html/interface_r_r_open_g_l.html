Detailed Description

A class to handle the OpenGL calls for an image processing application.

A single texture is used and it is operating on using GLSL (shaders). The class is designed to process a UIImage of a byte array image on the graphics chip easily.

Author
Stuart Lynch stuar.nosp@m.t.ly.nosp@m.nch@u.nosp@m.ea.a.nosp@m.c.uk
